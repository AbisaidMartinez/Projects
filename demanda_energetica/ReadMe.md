# Forecasting with SARIMA

This project consists of training, validating, and testing 
a time series forecasting model using SARIMA capable of
capturing variations in trends such as seasonal patterns
in the series.

To incorporate this model, real data on electricity demand
in California, USA, will be used.
The objective of the project is to develop a short-term predictive model
(max 24 hours into the future) based on historical data
to anticipate energy demand.

--------------------------------


Este proyecto consta del entrenamiento, validacion y prueba de 
un modelo de pronosticos de Series de Tiempo usando SARIMA capaz
de capturar variaciones en la tendencia como patrones estacionales
en la serie

Para incorporar este modelo se incorporar datos reales de demanda
energetica electrica de CA, EEUU.
El objetivo del proyecto consiste en desarrollar un modelo predictivo
de corto plazo (max 24 hrs a futuro), a partir de datos historicos
para anticipar la demanda energetica
